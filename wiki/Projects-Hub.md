# Official Plugins

### ExtractorPlugin

> This plugin can extract the info, search, and play songs directly from its source.

- [@distune/plugin-bandcamp](https://www.npmjs.com/package/@distune/plugin-bandcamp): Supports BandCamp, has song and album search.

### PlayableExtractorPlugin

> This plugin can extract and play songs from supported links, but cannot search for songs.

- [@distune/plugin-yt-dlp](https://www.npmjs.com/package/@distune/plugin-yt-dlp): Uses `yt-dlp` for supporting 700+ sites.
- [@distune/plugin-direct](https://www.npmjs.com/package/@distune/plugin-direct): Supports direct audio links.

> [!NOTE]
> To add your plugins/bots to this list, please [create a pull request](https://github.com/DisTuneJS/DisTune/pulls) with the edited [`wiki/Projects-Hub.md`](https://github.com/DisTuneJS/DisTune/blob/main/wiki/Projects-Hub.md) file.
